# 新機能: 最大連荘数と最大獲得枚数の追跡

`terminal_graph_simulator.py` スクリプトに、シミュレーション中に発生した最大の連荘数と、その連荘で獲得した最大のメダル枚数を追跡する機能が追加されました。シミュレーションの最後に、これらの統計が表示されます。

*   **最大連荘数:** 一度の天国モード（ドキドキ、超ドキドキを含む）滞在中に当選したボーナスの最大回数です。
*   **最大連荘獲得枚数:** 上記の最大の連荘中に獲得した合計メダル枚数です。

**使用方法:**

以前と同様に `terminal_graph_simulator.py` を実行するだけで、これらの新しい統計情報が自動的に計算され、シミュレーション結果の最後に表示されます。

**コマンド例:**

設定6で10,000ゲームのシミュレーションを実行し、結果とグラフ、そして最大連荘の統計を確認する場合：

```bash
python3 terminal_graph_simulator.py 10000 6
```

このコマンドを実行すると、通常のシミュレーション結果とクレジット履歴のグラフに加えて、以下のような情報が出力に追加されます。

```
最大連荘数: XX回
最大連荘獲得枚数: XXXX枚
```
